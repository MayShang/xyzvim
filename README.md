An automatic installation for vim and Package
===============================================

## installation

- version required

    `ubuntu16.04` and above

- install xyzvim

    ```bash
    git clone https://github.com/MayShang/xyzvim.git ~/.xyzvim
    cd ~/.xyzvim
    ./install.sh
    ```

## customize

change `~/.vimrc.local`, which will override `~/.vimrc`.
